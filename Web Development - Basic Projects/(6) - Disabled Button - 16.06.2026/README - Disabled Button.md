# Disabled Button

A Disabled Button is a button that cannot be clicked or interacted with by the user.

## Features

* Prevents user interaction
* Indicates unavailable actions
* Easy to implement using the `disabled` attribute
* Commonly used in forms and validation systems

## Files

* index.html
* style.css

## Example

Disabled Button

## Usage

Use a Disabled Button when an action should not be available until certain conditions are met.

Example:

* Form not completed
* Required fields missing
* Waiting for user input
* Processing another action
