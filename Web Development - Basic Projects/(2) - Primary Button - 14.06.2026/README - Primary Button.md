# Primary Button

## Description

Primary Button হলো একটি গুরুত্বপূর্ণ Action Button যা ওয়েবসাইটে Login, Sign Up, Submit, Save বা Get Started-এর মতো কাজের জন্য ব্যবহার করা হয়।

## Technologies

* HTML
* CSS

## Features

* Blue Background
* White Text
* Hover Effect
* Active Effect
* Smooth Transition

## Difficulty

Beginner
