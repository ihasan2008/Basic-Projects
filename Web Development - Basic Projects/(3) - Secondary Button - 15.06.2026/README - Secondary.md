# Secondary Button

## Overview

Secondary Button একটি সহায়ক (Secondary) Action Button। এটি সাধারণত এমন কাজের জন্য ব্যবহৃত হয় যা Primary Action-এর তুলনায় কম গুরুত্বপূর্ণ।

### Examples

* Cancel
* Back
* Close
* Skip
* Reset

## Technologies

* HTML5
* CSS3

## Features

* Modern Gray Design
* Rounded Corners
* Hover Effect
* Click Effect
* Smooth Animation

## Learning Objectives

এই Project থেকে শিখবে:

* HTML Button Element
* CSS Button Styling
* Hover State
* Active State
* CSS Transition
