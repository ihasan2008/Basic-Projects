# Gradient Button

## Overview

Gradient Button হলো একটি Modern UI Button যেখানে একাধিক রঙের Gradient Background ব্যবহার করা হয়। Hover করলে Gradient Animation এবং Glow Effect দেখা যায়।

## Technologies

* HTML5
* CSS3

## Features

* Linear Gradient Background
* Hover Animation
* Glow Effect
* Click Animation
* Modern UI Design
* Responsive Design

## Learning Objectives

* CSS Gradient
* CSS Animation
* CSS Transition
* Box Shadow
* Transform
