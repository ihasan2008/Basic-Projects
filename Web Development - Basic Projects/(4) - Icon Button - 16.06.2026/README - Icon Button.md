# Icon Button

A simple Icon Button component built using HTML and CSS.

## Features

* Clean design
* Rounded corners
* Hover effect
* Easy to customize
* Lightweight

## Files

* index.html
* style.css

## Example Icons

* 🔍 Search
* 🏠 Home
* ⚙️ Settings
* 👤 Profile

## Usage

Open `index.html` in your browser to see the Icon Button.
