# Simple Button Project

## 1. What is a Simple Button

Simple Button হলো একটি basic UI component যেখানে শুধু একটি clickable button তৈরি করা হয়।
এটা Web Development শেখার প্রথম ধাপগুলোর একটি।

---

## 2. Objective

এই project এর লক্ষ্য

- HTML দিয়ে button তৈরি করা
- CSS দিয়ে style করা
- Basic hover effect শেখা
- Clickable element বোঝা

---

## 3. Basic Simple Button

### HTML
```html
<button>Click Me</button>
```
